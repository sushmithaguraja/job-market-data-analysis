Overview

An end-to-end data science project that analyzes real-world job postings to identify trends in in-demand skills, salary vs experience, and hiring locations, presented through an interactive dashboard.

# What This Project Does

Analyzes job market data from public datasets

Identifies top skills for specific job roles

Compares average salaries by experience level

Highlights high-demand job locations

# Key Insights

Python and SQL are the most common skills across data roles

Senior-level positions offer significantly higher salaries

Major tech hubs dominate job demand, with notable growth in emerging locations
